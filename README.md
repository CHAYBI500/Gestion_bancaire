## Fonctionnalités principales
- Gestion des clients
- Gestion des comptes bancaires
- Opérations financières (dépôt, retrait, virement)
- Authentification et gestion des sessions
- Module IA pour l’aide à la décision sur les prêts bancaires

## Architecture du projet
- Backend : Java EE (JSP, Servlets, JPA, Hibernate)
- Base de données : MySQL
- Module IA : Python (Flask, Machine Learning)
- Communication : API REST

## Structure du dépôt
- backend-jee/ : application Java EE
- ai-module/ : API Flask et modèles IA

## Lancement du projet
1. Configurer la base de données
2. Déployer le backend sur Apache Tomcat
3. Lancer l’API Flask
